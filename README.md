# タブレット通販サイト

HAL専門学校の卒業制作として制作した、タブレット通販サイトです。

<img width="1185" height="567" alt="image" src="https://github.com/user-attachments/assets/161eb150-ea73-4de1-968f-7a56d62f52a1" />

## 使用技術

- PHP
- HTML
- CSS
- JavaScript
- MySQL

## 主な機能

- タブレット商品の一覧・詳細表示
- 商品カテゴリー別のページ表示
- ユーザーの新規登録・ログイン機能
- ショッピングカート機能
- 注文内容・購入金額の確認
- 郵便番号検索による住所入力補助
- PHP・MySQLを利用したデータ処理

## 実行環境

- PHPが動作するWebサーバー
- MySQL
- ローカル開発環境（例：XAMPP）
